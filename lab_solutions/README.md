# Lab Solutions

This folder is the single source of truth for lesson-by-lesson reference solutions.

- Each `NN_solution-...md` file contains the smallest runnable reference for that lesson.
- Use these files as both the solution archive and the guided code reference during class.
- The former `live_coding` folder was removed because it duplicated these materials without adding unique code logic.

Course rule: keep `lab_solutions` short, runnable, and focused on one clear edge case.
